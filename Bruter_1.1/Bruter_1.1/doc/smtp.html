Bruter :: SMTP

The SMTP module tests accounts against SMTP service.

Option(s)

Generally, you can use "Auto Detect" button to detect all options. Before using "Auto Detect" button, the target field (in the main dialog) must be validated.

  • Authentication Method: Method for authentication on SMTP service. Usually, you can use "Auto Detect" button to select available methods.
  • Use STARTTLS: This option determines whether STARTTLS command will be used or not. If you do not know about this command, we recommend you to use "Auto Detect" button.
    Remark: This option can be used only if useSSL (in the main dialog) is not checked.
  • Use Domain: If this option is checked, the program will use domain along with username. If the "Authentication Method" is not NTLM, the domain will be appended to username (username@domain).
    Remark: "Auto Detect" cannot detect this option).
  • Domain: This field is used only when "Use Domain" is checked.
  • Auto Detect: This button is used for auto-detect options in this dialog except "Use Domain". Before using this button, you need to input target information in the main dialog.
    Remark: You should double-check domain name after using "Auto Detect" button because it might be wrong.
