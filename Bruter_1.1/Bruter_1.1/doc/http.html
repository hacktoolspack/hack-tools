Bruter :: HTTP

The HTTP module tests accounts against HTTP authentication. It currently supports Basic, Digest and NTLM authetication method. The application uses HTTP keep alive method. The application knows when connections have to be disconnected.

Note: "Max Attempt/Connection" can be set to 0.

Option(s)

  • URL: The URL that requires HTTP Basic authentication.
  • Authen Method: The authentication method to be used. Normally, it is able to automatically choose with "Auto Detect" after URL is filled.
