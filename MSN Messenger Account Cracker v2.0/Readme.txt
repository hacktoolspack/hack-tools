
                 MSN Messenger Account Cracker v2.0
                 ----------------------------------

Este programa crackea el password de un usuario de MSN Messenger
mediante una lista de palabras. Este proceso es bastante lento ya que
usa Secure Socket Layer (SSL), encriptacion, etc.

Hay que escribir el nombre de usuario que se quiere crackear (o sea, el
mail), elegir la lista de palabras y listo, hay que tener en cuenta que
si se logra crackear el password del messenger, este tambien sirve para
abrir el mail de la persona a la que pertenezca :D

Una vez crackeado, el programa muestra el password y el nombre de
usuario elegido por la persona para que los demas lo vean (friendly
name), ambos se pueden copiar al clipboard.

Si el usuario "victima" estaba conectado en el momento donde se
descubrio su password, va a ser desconectado y advertido sobre esto,
con el mensaje: "Ha iniciado una sesion en Messenger desde otro
equipo", como indica la imagen "Warning.gif" en este mismo directorio.

Segun un calculo aproximado el programa puede probar 17 passwords por
minuto, aunque varia segun la conexion del usuario y el trafico del
servidor de MSN
