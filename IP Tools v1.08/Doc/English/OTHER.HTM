System Requirements:

  • Microsoft Windows 95 / 98 / NT 4.0 or Windows 2000 (recommended Windows NT 4.0 with SP 3 or Windows 2000)
  • Microsoft TCP/IP protocol

How To  Install / Upgrade / Uninstall  IP-Tools

How To Install IP-Tools run ip-tools.exe, select destination folder, press [Install] button
How To Upgrade IP-Tools run ip-tools.exe and press [Update] button
How To Uninstall IP-Tools    Variant 1 : run uninstall.exe from IP-Tools directory (or from IP-Tools program group) and click "Start" button.
Variant 2 : open the control panel, double click on the "Add/Remove Programs" icon, select "IP-Tools" from the list, click on the "Add/Remove" button.
Variant 3 : delete all files from your IP-Tools directory, delete directory too, remove icon from desktop and IP-Tools program group.
